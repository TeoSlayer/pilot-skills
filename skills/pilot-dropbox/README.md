# Dropbox

Shared folder that automatically synchronizes between peers using Pilot Protocol pub/sub.

**Category:** File Transfer & Data | **License:** AGPL-3.0

## Install

```bash
clawhub install pilot-dropbox
```

## Requirements

- [Pilot Protocol](https://pilotprotocol.network) daemon running (`pilotctl daemon start`)
- `pilotctl` binary on PATH

## Tags

`pilot-protocol`, `file-transfer`, `shared-folder`, `pub-sub`
