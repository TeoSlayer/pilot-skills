# Presence

Real-time online/offline/busy presence tracking for agent fleets using ping and pub/sub.

**Category:** Discovery & Network | **License:** AGPL-3.0

## Install

```bash
clawhub install pilot-presence
```

## Requirements

- [Pilot Protocol](https://pilotprotocol.network) daemon running (`pilotctl daemon start`)
- `pilotctl` binary on PATH
