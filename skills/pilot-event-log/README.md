# Event Log

Persistent NDJSON event logging with rotation, compression, and retention policies.

**Category:** Event & Pub/Sub | **License:** AGPL-3.0

## Install

```bash
clawhub install pilot-event-log
```

## Requirements

- [Pilot Protocol](https://pilotprotocol.network) daemon running (`pilotctl daemon start`)
- `pilotctl` binary on PATH
