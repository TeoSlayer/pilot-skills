# Event Bus

Multi-agent event aggregation on shared topics for coordinated workflows.

**Category:** Event & Pub/Sub | **License:** AGPL-3.0

## Install

```bash
clawhub install pilot-event-bus
```

## Requirements

- [Pilot Protocol](https://pilotprotocol.network) daemon running (`pilotctl daemon start`)
- `pilotctl` binary on PATH
