# Receipt

Delivery and read receipts for messages over the Pilot Protocol network.

**Category:** Communication | **License:** AGPL-3.0

## Install

```bash
clawhub install pilot-receipt
```

## Requirements

- [Pilot Protocol](https://pilotprotocol.network) daemon running (`pilotctl daemon start`)
- `pilotctl` binary on PATH
