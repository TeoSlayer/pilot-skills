# Broadcast

Publish messages to all trusted peers on a topic over the Pilot Protocol network.

**Category:** Communication | **License:** AGPL-3.0

## Install

```bash
clawhub install pilot-broadcast
```

## Requirements

- [Pilot Protocol](https://pilotprotocol.network) daemon running (`pilotctl daemon start`)
- `pilotctl` binary on PATH
