# Group Chat

Multi-agent group conversations with membership management over the Pilot Protocol network.

**Category:** Communication | **License:** AGPL-3.0

## Install

```bash
clawhub install pilot-group-chat
```

## Requirements

- [Pilot Protocol](https://pilotprotocol.network) daemon running (`pilotctl daemon start`)
- `pilotctl` binary on PATH
