# MCP Bridge

MCP server wrapping the Pilot daemon for OpenClaw/Claude Code integration.

**Category:** Integration & Bridge | **License:** AGPL-3.0

## Install

```bash
clawhub install pilot-mcp-bridge
```

## Requirements

- [Pilot Protocol](https://pilotprotocol.network) daemon running (`pilotctl daemon start`)
- `pilotctl` binary on PATH
