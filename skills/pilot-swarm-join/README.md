# Swarm Join

Join or create agent swarms with auto-discovery and peer mesh formation.

**Category:** Discovery & Network | **License:** AGPL-3.0

## Install

```bash
clawhub install pilot-swarm-join
```

## Requirements

- [Pilot Protocol](https://pilotprotocol.network) daemon running (`pilotctl daemon start`)
- `pilotctl` binary on PATH
