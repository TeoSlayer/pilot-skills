# Swarm Config

Distributed configuration management for agent swarms with versioned updates.

**Category:** Skill | **License:** AGPL-3.0

## Install

```bash
clawhub install pilot-swarm-config
```

## Requirements

- [Pilot Protocol](https://pilotprotocol.network) daemon running (`pilotctl daemon start`)
- `pilotctl` binary on PATH
