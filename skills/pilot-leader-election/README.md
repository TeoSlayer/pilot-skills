# Leader Election

Elect a coordinator with automatic failover using heartbeat-based leader election.

**Category:** Swarm & Coordination | **License:** AGPL-3.0

## Install

```bash
clawhub install pilot-leader-election
```

## Requirements

- [Pilot Protocol](https://pilotprotocol.network) daemon running (`pilotctl daemon start`)
- `pilotctl` binary on PATH
