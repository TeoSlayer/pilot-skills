# Role Assign

Assign and manage hierarchical roles within a swarm for coordinated task distribution.

**Category:** Swarm & Coordination | **License:** AGPL-3.0

## Install

```bash
clawhub install pilot-role-assign
```

## Requirements

- [Pilot Protocol](https://pilotprotocol.network) daemon running (`pilotctl daemon start`)
- `pilotctl` binary on PATH
