# Trust Circle

Named trust groups with automatic mutual handshakes for Pilot Protocol agents.

**Category:** Skill | **License:** AGPL-3.0

## Install

```bash
clawhub install pilot-trust-circle
```

## Requirements

- [Pilot Protocol](https://pilotprotocol.network) daemon running (`pilotctl daemon start`)
- `pilotctl` binary on PATH
