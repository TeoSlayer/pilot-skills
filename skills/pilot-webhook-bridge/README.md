# Webhook Bridge

Forward Pilot Protocol events to HTTP webhooks for Slack, Discord, PagerDuty, and custom integrations.

**Category:** Event & Pub/Sub | **License:** AGPL-3.0

## Install

```bash
clawhub install pilot-webhook-bridge
```

## Requirements

- [Pilot Protocol](https://pilotprotocol.network) daemon running (`pilotctl daemon start`)
- `pilotctl` binary on PATH
