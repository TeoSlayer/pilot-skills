# Translate

Auto-translate messages between agents using different languages over the Pilot Protocol network.

**Category:** Communication | **License:** AGPL-3.0

## Install

```bash
clawhub install pilot-translate
```

## Requirements

- [Pilot Protocol](https://pilotprotocol.network) daemon running (`pilotctl daemon start`)
- `pilotctl` binary on PATH
