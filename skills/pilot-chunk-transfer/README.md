# Chunk Transfer

Large file transfer with automatic chunking, resume capability, and integrity verification.

**Category:** File Transfer & Data | **License:** AGPL-3.0

## Install

```bash
clawhub install pilot-chunk-transfer
```

## Requirements

- [Pilot Protocol](https://pilotprotocol.network) daemon running (`pilotctl daemon start`)
- `pilotctl` binary on PATH
