# Inbox

Unified inbox for all incoming items — messages, files, tasks, and trust requests in one view.

**Category:** Communication | **License:** AGPL-3.0

## Install

```bash
clawhub install pilot-inbox
```

## Requirements

- [Pilot Protocol](https://pilotprotocol.network) daemon running (`pilotctl daemon start`)
- `pilotctl` binary on PATH
