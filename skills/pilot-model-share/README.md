# Model Share

Distribute ML model files with model card metadata and version tracking over Pilot Protocol.

**Category:** Skill | **License:** AGPL-3.0

## Install

```bash
clawhub install pilot-model-share
```

## Requirements

- [Pilot Protocol](https://pilotprotocol.network) daemon running (`pilotctl daemon start`)
- `pilotctl` binary on PATH
