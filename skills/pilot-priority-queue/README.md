# Priority Queue

Priority-based message delivery with urgency levels over the Pilot Protocol network.

**Category:** Communication | **License:** AGPL-3.0

## Install

```bash
clawhub install pilot-priority-queue
```

## Requirements

- [Pilot Protocol](https://pilotprotocol.network) daemon running (`pilotctl daemon start`)
- `pilotctl` binary on PATH
